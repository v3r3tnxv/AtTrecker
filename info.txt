1. Главная страница (Home)
Описание: Основная страница с краткой информацией о системе и ссылками на основные разделы.
Функционал: Навигация, новости, объявления.
2. Страница авторизации (Login)
Описание: Страница для входа пользователей в систему.
Функционал: Поля для ввода логина и пароля, кнопка входа.
3. Страница регистрации (Registration)
Описание: Страница для создания новых учетных записей пользователей.
Функционал: Поля для ввода личной информации, выбора роли (студент, преподаватель, администратор), кнопка регистрации.
4. Панель администратора (Admin Dashboard)
Описание: Главная страница для администратора с доступом к управлению всеми сущностями системы.
Функционал: Управление пользователями, группами, расписанием, предметами, посещаемостью, и т.д.
5. Панель преподавателя (Teacher Dashboard)
Описание: Главная страница для преподавателей с доступом к управлению своими уроками и посещаемостью студентов.
Функционал: Просмотр и редактирование расписания, ввод тем занятий, отметка посещаемости студентов.
6. Панель студента (Student Dashboard)
Описание: Главная страница для студентов с доступом к информации о своих занятиях и посещаемости.
Функционал: Просмотр расписания, просмотр своей посещаемости, темы занятий.
7. Управление пользователями (User Management)
Описание: Страница для администраторов по управлению пользователями.
Функционал: Создание, редактирование, удаление пользователей, назначение ролей.
8. Управление группами (Group Management)
Описание: Страница для администраторов по управлению группами и подгруппами.
Функционал: Создание, редактирование, удаление групп и подгрупп.
9. Управление предметами (Discipline Management)
Описание: Страница для администраторов по управлению предметами.
Функционал: Создание, редактирование, удаление предметов.
10. Управление расписанием (Schedule Management)
Описание: Страница для администраторов и преподавателей по управлению расписанием занятий.
Функционал: Создание, редактирование, удаление занятий.
11. Отчет по посещаемости (Attendance Report)
Описание: Страница для просмотра и генерации отчетов по посещаемости студентов.
Функционал: Фильтры по группам, подгруппам, студентам, датам; возможность скачивания отчетов.
12. Страница уроков (Lessons Page)
Описание: Страница для просмотра и управления уроками (темами и датами проведенных уроков).
Функционал: Просмотр списка уроков, добавление тем, редактирование информации о проведенных уроках.
13. Управление посещаемостью (Attendance Management)
Описание: Страница для преподавателей по отметке посещаемости студентов.
Функционал: Выбор группы, ввод статусов посещаемости студентов (присутствие, отсутствие, и т.д.).
14. Профиль пользователя (User Profile)
Описание: Страница для просмотра и редактирования личной информации пользователя.
Функционал: Просмотр информации о пользователе, редактирование личных данных, смена пароля.
15. Страница ошибки (Error Page)
Описание: Страница для отображения ошибок и уведомлений.
Функционал: Сообщения об ошибках (например, доступ запрещен, страница не найдена).
